* jQuery 사용하기

우리는 jQuery를 사용해서 DOM을 다룰 예정
jQuery코드가 문서상 모든 요소들이 전부 다 로드된 후에 실행할 수 있게끔 만들어 주어야항 => window.onload 

        onload : 페이지가 전부 다 로딩되고나면 자동으로 "딱 한 번 수행"
        해당 HTML무넛의 요소들도 전부 다 로딩되고 연동되어있는 외부 문서들도
        전부 다 로딩된 후에 onload속성에 대입한 이벤트 핸들러가 수행되는 특징을 가짐
    
        - jQuery에서는 해당 내용을 세 가지 구문을 제공하여 해결

        1)  
        jQuery(document).ready(function(){
            실행하고싶은 내용;
        });

        2)  $: jQuery키워드를 나타냄
        $(document).ready(function(){
            실행하고싶은 내용;
        });

        3)  
        $(function(){
            실행하고싶은 내용;
        });

        jQuery에서 해당 function은 HTML문서안에 있는 DOM요소들이 모두 로딩되면 바로 실행!
